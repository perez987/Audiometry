# Audiometry

## App for macOS 13+

<img src="Appicon.png" width="128px">

- Xcode 15.2 project

- Implements:
 	- patient data entry
 	- audiometric test input
 	- hearing loss assessment calculations
 	- including SAL and ELI indices with results display.
